

### 1. 修改表注释

修改表的注释的语法格式如下所示：
```SQL
ALTER TABLE table_name SET COMMENT 'tbl comment';
```
修改表注释:
```SQL
ALTER TABLE ads_app_channel_views_1d SET TBLPROPERTIES('comment' = 'App栏目每日流量统计');
```
### 2. 修改列注释
```sql
ALTER TABLE dim_org_td CHANGE deleted deleted STRING COMMENT '是否有效组织, 1-无效,0-有效';
```
